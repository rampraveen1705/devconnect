# DevConnect 🚀

A full-stack Firebase-based social app where users can:
- 🔐 Sign up / Login
- ✍️ Create posts
- 🔄 See real-time feed updates

## Tech Stack
- React
- Firebase Authentication
- Firestore Database

## Features
- Real-time updates using Firestore
- User-based posts
- Clean UI

## Setup

```bash
npm install
npm start
